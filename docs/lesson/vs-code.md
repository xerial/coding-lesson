Visual Studio Code
==================

## Keyboard Shortcut

マウス、トラックパッドを使わずに操作するのがプロの技。

- [Key Bindings for Visual Studio Code](https://code.visualstudio.com/docs/getstarted/keybindings#_keyboard-shortcuts-reference)


- ESC: Cancel 
- Moving Cursor
   - Up: Ctrl + P
   - Down: Ctrl + N
   - Forward: Ctrl + F
   - Backward: Ctrl + B
- Copy & Paste
  - Copy: Ctrl + C
  - Paste: Ctrl + V

- Find text 
  - Find: Cmd + F
    - Next: Enter
    - Previous: Shift + Enter

- Find File: Cmd + P
- Save File: Cmd + S


- Zen Mode
